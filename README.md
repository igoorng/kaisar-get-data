# kaisar-get-data

批量获取kaisai账号积分，账号信息保存到 kaisar_data.xlsx

账号登录失败把账号密码保存到 account_error.txt

# 启动脚本

1、修改email.txt文件，添加邮箱账号，一行一个账号

    `格式：邮箱账号----密码`

2、安装所需要的库

    `pip install -r requirements.txt`
3、运行脚本

    `python main.py`
